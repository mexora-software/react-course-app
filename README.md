# React Academy

App tipo curso online hecha con React + Vite.

## Qué incluye

- Módulos sobre fundamentos de React
- Componentes y props
- State y eventos
- Effects
- Routing
- Estado global
- Ejemplos de código
- Mini quiz interactivo
- Diseño responsive

## Cómo correrla

```bash
npm install
npm run dev
```

Después abre la URL que aparece en la terminal, normalmente:

```bash
http://localhost:5173
```

## Build de producción

```bash
npm run build
npm run preview
```
